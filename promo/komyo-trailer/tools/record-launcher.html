Record it (OBS · Window Capture):

1. Click a game above — a portrait window opens on the live site. Press ⛶ fullscreen in the game to drop the nav (optional — the template crops it anyway). Pick your mode in the start menu.

2. In OBS set Base + Output = 1080×1920, FPS 60. Add a Window Capture of this popup, right-click → Transform → Fit to screen, Alt-drag to crop chrome.

3. Play a few rounds, record continuously. If your backing size below isn't exactly 1080×1920, downscale after (never up): ffmpeg -i in.mov -vf scale=1080:1920 out.mp4.

4. Drop the mp4 in assets/footage/ and swap the <video src> in the template.
