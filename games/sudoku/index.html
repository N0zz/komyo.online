Sudoku

Fill the 9×9 so every row, column and box holds 1–9 — daily puzzles, hints and zen mode.

How to play

Fill the 9×9 grid so every row, column and 3×3 box holds the digits 1–9 exactly once. Tap a cell and enter a digit, use pencil marks for candidates, and ask for a hint when stuck — it highlights the reasoning instead of revealing the answer. Daily serves everyone the same puzzle; Zen is untimed practice at any difficulty.
